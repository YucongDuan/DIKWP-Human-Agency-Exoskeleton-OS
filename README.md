# DIKWP Human Agency Exoskeleton OS

A local-first research and product-design package for retaining human purpose, judgment, rights, relationships and future options in the agentic AI era.

## Quick start
1. Open `prototype/index.html`.
2. Read `docs/00_README_and_Delivery_Map.docx`.
3. Complete `templates/personal_ai_constitution.md`.
4. Use the Excel matrix to register real AI exposures.
5. Run `python src/humanagency_demo.py`.

## Boundary
This is not medical, psychological, legal, investment or employment advice. Metrics are self-reflection heuristics and must never be used for social scoring or access decisions.
